## 🚀 使用方法

### 1. 下载 M3U 文件
点击网站上的下载按钮，或直接访问对应的 `.m3u` 文件。

### 2. 用播放器打开
支持以下播放器：
- **Windows**: PotPlayer、VLC
- **Mac**: VLC、IINA
- **iOS**: nPlayer、Infuse
- **Android**: VLC、MX Player
- **电视**: KODI、TVBox、IPTV Smarters

### 3. 导入到设备
- 电视盒子：将 M3U 文件导入到 IPTV 应用
- 智能电视：通过 USB 或网络导入

## ⚠️ 注意事项

1. 部分频道可能需要科学上网才能正常播放
2. 直播源稳定性取决于源服务器
3. 部分线路可能需要切换才能正常播放
4. 建议使用 IPTV Checker 工具测试线路可用性
