## 🚀 使用方法

### 1. 下载 M3U 文件
点击网站上的下载按钮，或直接访问对应的 `.m3u` 文件。

### 2. 用播放器打开
支持以下播放器：
- **Windows**: PotPlayer、VLC
- **Mac**: VLC、IINA
- **iOS**: nPlayer、Infuse
- **Android**: VLC、MX Player
- **电视**: KODI、TVBox、IPTV Smarters

### 3. 导入到设备
- 电视盒子：将 M3U 文件导入到 IPTV 应用
- 智能电视：通过 USB 或网络导入

## 📱 使用教程

### Windows 用户
1. 下载 M3U 文件
2. 用 PotPlayer 或 VLC 打开
3. 文件 → 打开网络串流/文件

### Mac 用户
1. 下载 M3U 文件
2. 用 VLC 或 IINA 打开
3. 文件 → 打开文件

### iOS 用户
1. 下载 M3U 文件
2. 用 nPlayer 或 Infuse 打开
3. 添加播放列表

### Android 用户
1. 下载 M3U 文件
2. 用 VLC 或 MX Player 打开
3. 添加播放列表

### 电视盒子
1. 下载 M3U 文件
2. 用 KODI 或 TVBox 打开
3. 添加 IPTV 源

## ❓ 常见问题

### Q: 部分频道无法播放？
A: 部分频道需要科学上网，或尝试切换线路

### Q: 直播源多久更新一次？
A: vbskycn 每日更新，其他源不定期更新

### Q: 如何获取更多频道？
A: 访问源仓库获取最新直播源

### Q: 支持哪些播放器？
A: PotPlayer、VLC、nPlayer、KODI、IPTV Smarters、TVBox 等

## ⚠️ 注意事项

1. 部分频道可能需要科学上网才能正常播放
2. 直播源稳定性取决于源服务器
3. 部分线路可能需要切换才能正常播放
4. 建议使用 IPTV Checker 工具测试线路可用性
